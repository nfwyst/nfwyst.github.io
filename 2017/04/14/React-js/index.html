ReactJs

发表于 2017-04-14 | 分类于 前端

React 是一个 Facebook 开源的一套框架, 它只负责视图的渲染, 并在数据发生变化的时候, 重新渲染视图中的组件, 在 React 中视图是以组件的方式来组织的, 数据以流动的方式来传递, 而组件又有组件的生命周期, 在组件的不同的生命周期可以执行不同的钩子

组件的状态与重新渲染

React 使用 Virtual DOM 解决了组件重绘问题, 每当组件的 this.state 发生变化的时候(使用 this.setState 重新设置 state), React 就会计算虚拟DOM和真实 DOM之间的区别, 如果有必要, 就会重绘当前组件, 而不是重绘整个页面

组件生命周期

componentDidMount

组件在加载之后会触发 componentDidMount

componentWillReceiveProps

props是父组件传递给子组件的。父组件发生render的时候子组件就会调用componentWillReceiveProps

shouldComponentUpdate

组件挂载之后，每次调用setState后都会调用shouldComponentUpdate判断是否需要重新渲染组件。默认返回true，需要则重新渲染

componentWillUpdate

在组件重新渲染之前，会调用 componentWillUpdate

componentDidUpdate

除了首次组件渲染时调用的是 componentDidMount, 后续的渲染都是调用 componentDidUpdate, 表示渲染完成

componentWillUnmount

组件在被卸载之前会调用 componentWillUnmount, 一般情况下载 componentDidMount 里面注册的事件需要在这里删除

组件更新方式

  1. 首次渲染时 render
  2. 调用 this.setState 修改状态时 render
  3. 父组件发生更新时 render
  4. 调用 this.forceUpdate 时更新
