Sass

发表于 2017-03-07 | 分类于 前端

Sass 是一种 CSS 的预处理语言, 它在兼容 CSS 的同时, 增加了很多特性, 它的特性之一就是编译, 支持变量, 函数, 混淆, 导入, 运算, 能够自动生成 CSS , 节省了开发人员的时间, 发挥了计算机的本来作用

安装好之后, 可以在终端上打开交互式环境

Sass 的两种格式

Sass 有两种格式一种是 .sass, 一种是 .scss, 前者采用类似于 python 和 styl 的缩进语法, 后者采用一种更传统的易读方式, 而且后者(.scss)更流行一些, 可以看一下两者的对比:

指令

一般的指令也指关键字, sass 中的指令都用 @ 作为前缀, 像前面介绍的 @mixin, @function, @include, @return 都属于指令
除了以上指令, sass 中的很大一部分是用于解决复杂逻辑的控制指令, 包含:

  • @if
  • @for
  • @each
  • @while
  • @error

@if 指令用于条件判断, @for 用于循环, @each 用于遍历 list 并根据不同的 list 内容生成不同的样式, @while 比 @for 更灵活, @error 用于处理错误并输出错误
