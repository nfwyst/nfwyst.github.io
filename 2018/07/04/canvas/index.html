canvas 入门

发表于 2018-07-04 | 分类于 前端

什么是 canvas

canvas 是 HTML5 提供的新 API, 它是一个矩形区域的画布, 通过 API 我们可以在canvas上进行绘制, 通过绘制来控制每一个像素的显示.

创建 canvas

使用 canvas 首先需要创建 canvas 标签, 并设置标签属性

在 canvas 上绘图, 首先需要得到 canvas 的 2d 上下文

接下来就可以进行绘制了.

绘制直线

首先我们设置直线的路径, 先移动到坐标(100, 100), 绘制到(200, 100), 接着再绘制到(100, 200)

以上只是绘制了路径, 这里的路径实际上就是状态, 真正绘制则需要填充路径

绘制多边形

每一次lineTo都会产生一个路径, 如果我们需要画一个闭合的多边形, 可以使用ctx.closePath() 将起点和终点连接起来
比如绘制一个三角形

需要注意的是 只有 IE9+ 的浏览器版本才支持 canvas

描边与填充

描边

描边表示绘制直线, 在绘制之前可以设置以下属性指定绘制的样式

  • lineWidth(线宽, 以点为中线)
  • strokeStyle(颜色)

填充

填充表示用指定颜色(默认黑色)填充闭合路径, 它绘制的是一个闭合的区域, 在填充之前可以设置填充的样式

  • fillStyle(颜色)

    对于设置了 lineWidth 属性的描边, 如果同时设置了填充, 那么填充会将 lineWidth 的一半也进行覆盖, 比如 lineWidth 为 4 的描边, 填充时其路径内部的2像素将被填充覆盖掉

beginPath (状态)

canvas 是基于状态的, beginPath 表示新的状态开始, 上一次绘制状态结束, 同时当前状态可以继承上一次状态所设置的样式, 比如下面的例子中, 我们在第一个状态中绘制一条红色的直线, 然后在第二个状态中绘制一条绿色的直线

绘制矩形

ctx.rect(x, y, w, h) 可以绘制一个矩形路径, 然后可以进行描边或填充, 但也可以使用直接填充矩形或直接描边矩形
strokeRect(x, y, w, h) 是描边矩形, fillRect(x, y, w, h) 是填充矩形, x, y 表示矩形左上角的坐标, w, h 分别表示宽度和高度

绘制矩形后, 可以使用 clearRect(x, y, w, h) 清除矩形, 参数分别为起始 x 坐标, 起始 y 坐标, 矩形区域的宽度w, 矩形区域的高度h

弧与坐标角度

canvas 坐标默认以 x 轴正向为 0 度, 顺时针方向为正角, 逆时针方向为负角.
绘制弧使用 ctx.arc(圆心x坐标, 圆心y坐标, 半径, 开始弧度, 结束弧度, 是否逆时针绘制[默认为false]), 它的角度参数以弧度为单位, 弧度和角度有一个转换关系 1arc = deg * Math.PI / 180

绘制文字

绘制之前可以对文字设置如下属性:

  1. ctx.font(设置字体)
  2. ctx.textBaseline(设置文字和哪条线对齐)
  3. ctx.textAlign(设置字体的对齐方式)
  4. ctx.direction(用来描述绘制文本方向的属性, ltr 为从左到右, rtl 为从右到左)
    然后可以使用 fillText(‘content’, x, y), strokeText(‘content’, x, y) 进行填充或绘制, 其中(x, y)表示文字将被绘制的坐标位置

ctx.measureText(文本内容) 可以测量文本的宽度, 返回包含指定文本宽度的对象

绘制图片

绘制图片使用 drawImage(img, sx, sy, sw, sh, x, y, w, h) 方法, 参数分别是 图片对象, 裁剪的 x 坐标, 裁剪的 y 坐标, 裁剪图片的宽度, 裁剪图片的高度, x 坐标, y 坐标, 图片缩放宽度, 图片缩放高度

绘制帧动画

序列帧实际上就是用定时器不断地使用 drawImage 去绘制, 每一次都可以通过改变裁剪区域 sx, sy 坐标的值实现动画效果
