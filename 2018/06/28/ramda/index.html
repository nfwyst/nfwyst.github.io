ramda

发表于 2018-06-28 | 分类于 functional frontend

ramda 是一个 js 函数式编程库, 下面跟着文档来看一下, 体验一下函数式编程的乐趣吧

R.__

这是柯里化函数的参数占位符, 允许部分应用任何位置的参数, 与位置无关

R.add

这是一个基础函数, 它将两个数相加, 也是一个 curry 函数

R.addIndex

向列表迭代函数的回调函数添加两个新参数: 当前索引, 整个列表, 创建新的列表迭代函数, 来看看吧

比如迭代函数 map, 其回调函数只有一个参数, 如果需要传入索引作为另一个参数, 可以使用 addIndex 构造一个新的迭代函数

以上的 add 会接受到列表的每一个项目的值和项目的索引, 返回相加的值.

adjust

使用指定函数修改指定位置的值

all

返回列表中每个元素是否均满足断言的布尔值

allPass

返回给定参数是否满足所有的断言的布尔值

anyPass

返回给定参数是否满足一个断言的布尔值

always

返回一个函数, 这个函数始终返回一个固定的值

identity

将输入原样返回, 适合用作默认或者占位函数

and

当两个参数都是 true, 返回true, 其余情况返回 false

any

当列表中有一个元素满足断言, 返回 true, 否则返回 false

ap

ap 将函数列表作用于值列表上, 可以作为 S combinator

pipe

执行函数组合, 前面函数的输出是后面函数的输入, 第一个函数可以是任意元, 后面的函数必须是一元函数

apture

返回一个新的列表, 新列表的元素是原来列表相邻元素组成的 n 元组, 如果 n 的值大于原列表的长度则返回空列表

append

将一个元素添加到列表的末尾

prepend

将一个元素添加到列表的头部

apply

将函数作用于参数列表, 并且可以将变参函数转换为定参函数, 也可以指定上下文

applySpec

它接受一个属性值包含函数的对象, 返回一个能生成同样结构的对象的函数, 返回的函数使用传入的参数调用对象的函数属性, 并生成相应的属性值.

applyTo

将指定的值作用于指定的函数
